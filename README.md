# Shell Emulator

## Описание

**Shell Emulator** — это эмулятор командной оболочки на Python, который имитирует файловую систему, загруженную из TAR-архива. Пользователи могут перемещаться по виртуальной файловой системе, выполнять базовые команды, такие как `ls`, `cd`, и получать информацию о текущем пользователе. Оболочка работает с виртуальной файловой системой, которая загружается из указанного TAR-файла.

## Основные команды

- `ls` — выводит список файлов и папок в текущей директории.
- `cd <directory>` — Переход в директорию (cd /корневая директория).
- `cd ..` - Возврат в прошлую директорию
- `history` — Показать историю выполненных команд.
- `uniq` — Вывести уникальные строки из указанного файла.
- `tail` — Показать последние строки указанного файлa.

## Установка и запуск

1. Подготовьте TAR-архив с файловой системой (например, `test_filesystem.tar`), которая будет использоваться в эмуляторе.
2. Задайте два ключа в форме: python "Название ПК" "путь к вашему зипфалу.tar"
